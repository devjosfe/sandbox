📖 Memory Strategies Explained

Sliding Window: Sends only the last N messages. Good balance of context and token usage. Best for most cases.

Full History: Sends ALL messages. Complete context but hits token limits on long conversations.

Summary Memory: Old messages → LLM summary → cached. Best for very long conversations where old context matters.
